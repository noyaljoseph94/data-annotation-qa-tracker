# 📊 Daily QA Tracker — Data Annotation

> A professional Excel-based quality assurance tracking system for data annotation operations, built from real-world experience managing 1,000+ frames daily with 99%+ accuracy benchmarks.

---

## 📁 What's Inside

| File | Description |
|------|-------------|
| `QA_Tracker_Noyal_Joseph.xlsx` | Main tracker with 3 sheets |

### Sheet 1 — Daily QA Log
- Log daily production: frames assigned, completed, and QA reviewed
- Auto-calculated accuracy percentage per day
- Target met indicator with colour coding
- Weekly summary dashboard (total frames, avg accuracy, error count, days on target)

### Sheet 2 — Monthly Summary
- Week-by-week production breakdown
- Total frames and average accuracy per week
- Auto-calculated totals using Excel formulas

### Sheet 3 — Error Log
- Track every error found during QA review
- Classify by severity: Critical, High, Medium, Low
- Log resolution status and reviewer comments
- GDPR/PII violation tracking

## 🛠️ How to Use

1. Download `QA_Tracker_Noyal_Joseph.xlsx`
2. Open in Microsoft Excel or Google Sheets
3. Fill in your daily data in **Sheet 1 (Daily QA Log)**
4. Summary metrics auto-update in rows 11–12
5. Log any errors found in **Sheet 3 (Error Log)**
